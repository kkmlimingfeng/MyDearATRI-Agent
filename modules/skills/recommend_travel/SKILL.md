# 推荐旅游地点

当用户想了解某个城市或某种天气下适合去哪里玩、有哪些景点推荐时，使用本技能。

## 触发场景

- "北京有什么好玩的？"
- "苏州天气怎么样，适合去哪玩？"
- "下雨天适合去哪里旅游？"
- "推荐一些适合晴天去的景点"

## 工作流程

1. **查询天气**：使用 `get_weather(city)` 获取目标城市的实时天气。
2. **获取景点推荐**：根据返回的天气状况，使用 `get_attraction(city, weather)` 搜索合适的景点。
3. **综合回答**：结合天气信息和景点推荐，给出最终的旅行建议。

## 输出格式

必须严格遵循 ReAct 格式：

```text
Thought: [你的思考过程]
Action: [工具调用或 Finish]
```

## 示例

用户：北京现在天气如何，有什么地方可以玩？

```text
Thought: 用户想了解北京的天气和景点，我先查询北京当前天气。
Action: get_weather(city="北京")

Thought: 北京天气为附近有零星降雨，接下来查询适合雨天的景点推荐。
Action: get_attraction(city="北京", weather="附近有零星降雨")

Thought: 已获取天气和景点信息，可以给出最终建议。
Action: Finish[北京当前天气为附近有零星降雨，气温27摄氏度。推荐景点为故宫和中国国家博物馆，雨天参观室内景点体验更佳。]
```
